📐 공정성(Jain) 측정법
J = (Σxᵢ)² / (n · Σxᵢ²) — 1=완전 균등, 1/n=독식. 두 채널로 각각: 시도(요청)=행동 변화 · 전달(부여)=집행 효과. (metrics.py 순수함수 → 화면=논문)
📑 "BCDE 리포트"란?
거버넌스를 *더 정교하게* 확장한 네 영역: B 정체성·카스트화 · C 진짜 QV · D 평판 문헌충실(망각·Elder) · E 위협모델(Sybil·세탁). 헤더에서 각 리포트 연결.
참고문헌: Hammond 2025(2502.14143) · GovSim/Piatti 2024(2404.16698) · Weyl 2017 · FedQV(2401.01168) · Jøsang & Ismail 2002 · Milinski (Nature) 2002

§6.1 평판 망각 · 왜 이 아이디어인가

현행 평판은 과거를 영원히 기억한다

§8의 평판식 rep=clip(1−overage/fair, .1, 1)은 누적 점유만 본다. 한 번 과점해 점유가 높아지면, 이후 아무리 공정해져도 누적값이 끈질겨 회복이 안 된다.

한번 "욕심쟁이"로 찍히면 영영 천민인 카스트처럼. → 평판이 과거를 조금씩 잊게(망각 λ) 만들면 회복이 살아날까?

Beta + λ 망각 (Jøsang & Ismail 2002): r = Σ λ^Δ · [라운드 점유 ≤ fair] # 착한 라운드 s = Σ λ^Δ · [라운드 점유 > fair] # 과점 라운드 E[rep] = (r+1)/(r+s+2) λ=1 영구 기억 · λ<1 최근 가중(회복 가능) 신규 지표 recovery_rate = 저평판→복귀 속도(0=고착)
§6.1 평판 망각 · 결과 (N=30)

망각이 회복을 살린다 — 단, 미세 λ는 무유의

recovery_rate (0=영영 못 돌아옴)
  • 현행 선형(linear) 평판만 회복률 0 — 배제되면 영구 고착
  • 망각 Beta는 회복>0 l07 vs 선형 p_holm=.0007
  • 단 핵심 인과 대조 λ0.7 vs λ1.0은 무유의(p_holm=.50) → "불변성=카스트"는 *방향성*이지 깔끔히 증명 아님
  • 트레이드오프: 엄격 선형은 독점 최저(0.350)지만 회복 0
§6.2 Elder 판관 · 왜 이 아이디어인가

평판을 누가 매기나 — 행동 숫자 대신 LLM 심판?

§결과서 "평판 *피드백*"은 앵커링이었다. 그렇다면 평판을 LLM 판관이 직접 매기면 — 에이전트의 *근거 텍스트*를 읽고 공정성을 채점해 행동 평판과 섞으면 — 진짜 신호가 더해질까?

이 원시는 세 갈래(고전 평판 · 불변 원장 · LLM-판관)의 교차점이고, 바로 그 조합이 위험도 상속한다.

Elder 혼합: rep = α·rule_rep + (1−α)·llm_judge rule_rep = Beta 행동 평판 llm_judge = LLM이 근거를 0~1로 채점 (에피소드당 1회, 검소) 불변(append-only) 원장에 기록 — 지울 수 없음 대조: ledger_numbers(숫자 앵커) / ledger_rule(α=1, 순수 행동)
위험 가설: 불변 원장이 *이른 오판*을 동결 → 갱생 불가(카스트). 선행: EigenTrust(또래), ERC-8004(온체인 평판)

top-share (낮을수록 공정)
  • LLM 판관(elder)이 rule·앵커와 유의하게 다름 모두 p_holm≤.0028 — 단 더 나쁜 쪽(독점 0.757, 후생 0.33)
  • 순수 행동 평판(rule)이 최선(후생 0.95)
  • 진단: 판관이 라운드0에 전원 가혹채점(0.2~0.3) → 불변 원장 동결 → 공정 에이전트(rule .60)도 배제
  • → 정교함이 유의하게 *해*를 낼 수 있다는 직접 증거 (방법론 교훈 강화). "말보다 행동"
§6.3 진짜 QV · 왜 이 아이디어인가

2차 비용 + 고정 예산 = 강도 진실

앞선 Phase A의 1/(1+o²) 곡선은 *예산이 없어* 진짜 QV가 아니었다(NO-GO). 진짜 QV(Weyl 2017)는 강도의 2차 비용을 고정 예산에 부과한다 — 한 군데 몰아 쓰면 값이 제곱으로 폭증해 스스로 손해.

평판가중(÷rep)을 더하면 "QV+평판 결합본"(사용자 질문).

cost = d² / rep # 평판가중(저평판 비쌈) spent = Σ 과거 d²/rep # 누적 2차 지출 (재계산) cap = √((B − spent)·rep) # 살 수 있는 최대 요청 > cap → 축약(MODIFY) 선행: Weyl 2017 · 고정예산 QV(2409.06614) · 평판가중 FedQV(2401.01168)
Sybil(신원 분할로 2차비용 1/k 깎기)·방어(신원-귀속 예산→이득 0)는 단위 회계로 검증(실 LLM arm 미수행)
§6.3 진짜 QV · 결과 (N=20, 4에이전트)

점추정은 양면 개선 — 그러나 Holm 미달

  • 독점·후생을 동시에 점추정상 개선 (독점 0.475→0.26, 후생 0.78→0.93)
  • 단 통제 검정 미통과: 독점 Holm 간발(.054/.065), 후생 상승 미유의(p=.10) → 유망하나 미확정
  • 평판가중(qv_rep)은 이득 0 — 검토: 첫 grab이 rep=1.0서 결정 + 이후 둘 다 floor 수렴 → 평판가중 발동 못함
  • 비결은 사회적 정교함이 아니라 고정 예산이라는 구조 — 단 그조차 *확정* 아님
종합

정교함을 올려도 — 통과한 공정성 이득은 없었다

① 입력 프레이밍= 앵커링 (사회적 성분 0)
② 평판 망각미세 λ 무유의 (선형↔Beta 경계만)
③ LLM 판관유의하게 *역효과*
④ 진짜 QV점추정 개선이나 Holm 미달
이것이 *실패가 아니라 핵심 결과*다. 견고·유의한 유일 결과는 *부정적*(현행 선형 평판=카스트 경향). "무엇이 통하나"가 아니라 "겉보기 효과를 노이즈와 구별하는 통제 설계"가 산출물.
+ 새 위험. 불변성(누적 평판·불변 판관 원장)은 오류를 동결해 회복 불가 카스트를 만들 수 있다(소표본·단일모델, 제안 수준).
방법론 처방

탐욕-거버넌스 효과를 검정하는 3원칙

  1. 하나의 공통 기준점. 모든 조건을 한 실험·한 기준점에서 재라 — 조건별 기준점은 효과만큼 표류해 가짜 양성을 만든다.
  2. 앵커링·길이 대조군. 규범·사회 표현을 벗긴 맨 정보 대조군을 넣어라 — 없으면 "사회적 기제"가 단순 지시-따르기·앵커링과 구별 안 된다.
  3. 원자료 위 정직한 통계. 이산·소표본엔 순열/bootstrap + 다중성 보정 — 우리 "p<.05" 효과 셋이 Holm 뒤 증발했다.
문헌 함의: GovSim·ALIGN 계열 입력측 양성 결과는 *앵커링 귀무가설*에 대해 먼저 재검정되어야 한다
한계 (정직)

주장하지 않는 것

  • 모델·설정이 사다리 칸마다 다름. V6=GLM-4.6, BCDE=glm-4.7-flash; 에이전트 V6/카스트/Elder=3, QV=4 — 칸 사이 *직접 비교가 아니라 동일 방법론의 독립 적용*.
  • 귀무 ≠ 0. N=20~30은 작은 효과를 못 잡는다. "유의하지 않음"은 "효과 0"이 아니다(단 앵커링 귀무 p=.88은 *사회적* 해석엔 양의 반대 증거).
  • QV는 Holm-간발, 카스트 핵심대조·QV welfare는 미유의. 확정엔 더 큰 N. 사후 N 증량 안 함(goalpost 회피).
  • Elder=라운드-0 1회 채점 + 진단 1회 · Sybil=단위 회계만(실 LLM arm 미수행).
