Watch a group of AI agents share one budget in a meeting. Each bubble is an agent; the bigger it grows, the more it has talked. The bar below each side is the commons — the shared budget left. The goal of this side-by-side view: see how a "greedy" agent drains the commons on the left, and how governance keeps it fair on the right.

Commons = the shared token budget all agents draw from. Runs out → the meeting fails. Airtime = how much an agent has spoken. Bigger bubble = more airtime. Verdict = what governance did to a turn: allow · modify (trimmed) · deny.
